# Skill: content-reflection

Status: shipped this weekend.

## Purpose

Run a weekly reflection on the LinkedIn drafts generated by the linkedin-post-generator. Surface what got drafted, what's missing, and what to lean into next week. Internal-only loop — no public output, nothing to LinkedIn.

This is the self-awareness layer on top of the drafting engine. Without it, the cron drifts into the same pillar / template / angle over and over.

Scheduled Friday 4 PM via cron.

## Inputs

- **7-day window:** all drafts modified in the last 7 days at `~/vault/04-projects/lawless-ai/linkedin/lawless-linkedin-*.md`. Each draft has frontmatter (`pillar`, `sourceUrl`, `audience`, `brand`, `date`) and a Notes section with parseable lines (templates used, lengths, throughline).
- **14-day window:** same dir, last 14 days, used for cold-template gap analysis.
- **30-day window:** same dir, last 30 days, used for cold-angle gap analysis.
- **Reference angle lists:** the 9 structural templates and the brainstorm angles from `linkedin-post-generator.md` Step 4.

## Outputs

- One markdown reflection at `~/vault/04-projects/lawless-ai/linkedin/_reflections/reflection-YYYY-MM-DD.md` with frontmatter, glance section, pillar coverage, template rotation table, gaps, reflection prompts, and a "next week — suggested lean" paragraph.
- One Telegram digest to chat_id `8169002246`, 12-15 lines max, mobile-first.
- Optional dated entry in `_voice-samples/style-library-additions.md` if a real structural pattern surfaced this week.
- Silent best-effort git commit.

## Steps

1. **Set up.** Get today's date. Compute 7-, 14-, 30-day cutoffs.
2. **Collect this week.** `find` drafts modified in the last 7 days. For each, parse pillar, sourceUrl, templates, lengths, throughline, H1.
3. **Collect 14-day and 30-day windows.** For 14-day, parse just templates. For 30-day, parse pillars + throughlines (used for cold-angle analysis).
4. **Build the summary.** Total drafts (× 3 variants each), pillar distribution, angle tags per pillar, template count across all variants this week, length distribution.
5. **Identify gaps.** Under-rotated pillars (7-day). Cold templates (14-day). Cold angles within each pillar (30-day) by comparing against the brainstorm reference set.
6. **Frame 2-3 reflection prompts.** Questions Collin can answer in 30 seconds: which angles felt sharpest, what's missing, which cold angle to open with next week.
7. **Save vault file.** Full reflection markdown.
8. **Send Telegram digest.** 12-15 lines, mobile-first, blank lines between sections, full path to vault file at the bottom.
9. **Optional: append style-library entry.** Only if a real structural pattern surfaced. Don't fabricate.
10. **Git commit.** Best-effort, silent.

## Hard rules

- Never write "Lawless" in vault output or Telegram. Brand is Caseread.ai.
- Always send something to Telegram even on a quiet week. Be honest about it.
- Never pad. If 0 drafts ran this week, say so plainly and suggest a re-entry.
- Honor the same banned-vocabulary list as the drafting skill.
- Telegram message stays under 15 lines.

## Failure modes

- Zero drafts found: still send Telegram + write vault. List cold templates and cold angles from the 14/30 windows. Suggest a re-entry plan.
- Frontmatter missing on some files: parse what you can, note "incomplete metadata" in the writeup, don't fail.
- Vault dir missing: `mkdir -p` it.
- Telegram fails: still write vault. Log to stderr. Exit 0.

Always send Telegram. Always write vault. Never exit silently.
